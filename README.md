# SmartMeal 🍽️

SmartMeal is an AI-powered iOS app that helps users turn food photos into personalized recipes and meal plans. Built using **SwiftUI**, **Core ML**, and **custom vision models**, it makes healthy eating easier.

## 🚀 Features
- 📷 Scan ingredients using your camera or gallery
- 🧠 Analyze images with ML to detect food items
- 🍽️ Generate recipe suggestions instantly
- 🔔 Notification support for reminders
- 🧩 Modular, scalable MVVM architecture

## 🛠 Built With
- SwiftUI & Combine
- Core ML & Create ML
- Xcode
- Vision framework
- Firebase (optional)
- Git & GitHub
